Stripe Payment - Read Me

The Stripe payment API provides numerous testing tools for developers to evaluate their integration. Among these tools are a set of dummy card numbers specifically designed to test various card scenarios and responses. These scenarios include generic declines, insufficient funds, expired card, incorrect CVC decline, and incorrect number decline. You can find the full list by clicking here.
